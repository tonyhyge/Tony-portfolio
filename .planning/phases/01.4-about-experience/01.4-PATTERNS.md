## Shared Patterns

### Data File Pattern
**Source:** `src/app/data/skills.ts`, `src/app/data/research.ts`
**Apply to:** `src/app/data/experience.ts`

```typescript
export interface ModelName {
  // typed fields
}

export const dataArray: ModelName[] = [
  // typed entries
];
```

Export both the interface and the data array. Consumers import the data array for rendering, and optionally the interface for typing.

### Feature Component Directory Pattern
**Source:** `src/components/hero/`
**Apply to:** `src/components/about/`, `src/components/experience/`

```
feature-name/
â”śâ”€â”€ feature-name.tsx        # Parent composition component
â”śâ”€â”€ child-component.tsx      # Child presentational components
â””â”€â”€ (no CSS modules â€” all Tailwind)
```

- Parent component coordinates layout and data flow
- Child components handle individual UI elements
- All styling via Tailwind utility classes

### Section + Container Pattern
**Source:** `src/components/section.tsx`, `src/components/container.tsx`
**Apply to:** `page.tsx` integration

```typescript
<Section id="about" title="About">
  {/* content â€” no Container needed, Section provides it */}
</Section>
```

- Section provides: `id` (for scroll targeting), `title` (as heading), `aria-labelledby`, responsive padding
- Container is embedded inside Section â€” no need to wrap content manually
- Section acts as a `<section>` element with `py-16 md:py-24`

### Component Export Pattern (no React.FC)
**Source:** All existing components (Hero, Section, Container)
**Apply to:** All new components

```typescript
export function ComponentName({ prop1, prop2 }: Props) {
  return (/* JSX */);
}
```

- Use `export function`, not `export default function`
- No `React.FC` or `React.FC<Props>` wrapper
- Explicit prop interfaces defined at top of file or inline

### Badge Usage Pattern
**Source:** `src/components/ui/badge.tsx`
**Apply to:** Tech tags in `experience-entry.tsx`

```typescript
<Badge variant="secondary">{tag}</Badge>
```

- `variant="secondary"` recommended (good contrast in both light/dark modes, doesn't compete with accent-colored timeline dots)
- Tags container: `flex flex-wrap gap-1.5` for natural wrapping
