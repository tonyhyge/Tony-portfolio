# Phase 01.3 Plan 01: HexagonBadge Component Summary

**One-liner:** Decorative hexagon badge component with gradient cyan-to-violet border/glow, swappable to profile photo container.

## Objective

Create the HexagonBadge component — a decorative geometric hexagon placeholder for the profile photo, with gradient border/glow effect. Satisfies D-06 (hexagon badge with accent gradient border/glow), D-07 (right-side floating badge), D-08 (swappable to photo later).

## What Was Built

**src/components/hero/HexagonBadge.tsx** — Exported `HexagonBadge` React component.

### Key Features
- **Decorative mode:** Renders inline SVG with hexagon polygon, gradient stroke (`var(--accent)` → `var(--accent-secondary)`), SVG glow filter, semi-transparent card-colored fill
- **Image mode:** When `src` prop provided, renders `<img>` clipped to hexagon shape via `clipPath: polygon(...)`
- **Responsive sizing:** `size-[64px] md:size-[120px]` — 64px mobile, 120px desktop
- **Accessibility:** `aria-hidden="true"` when decorative, `alt` text required when image provided
- **Theming:** Uses CSS custom properties (`--accent`, `--accent-secondary`, `--color-card`) for theme-aware colors
