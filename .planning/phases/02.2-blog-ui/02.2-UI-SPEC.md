## Spacing Scale

Standard 4-point scale inherited from Tailwind + shadcn:

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Tag pill internal padding, compact gaps |
| md | 16px | Card internal padding, element gaps |
| lg | 24px | Section padding, card spacing in grid |
| xl | 32px | Layout gaps between major sections |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions: none.

## Typography

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Small | 14px (0.875rem) | 400 (regular) | 1.4 | Tags, metadata, breadcrumb, TOC items, date, reading time |
| Body | 16px (1rem) | 400 (regular) | 1.5 | Post excerpts, card descriptions, prose content |
| Heading | 20px (1.25rem) | 600 (semibold) | 1.3 | Card titles, section headings ("All Posts"), series banner |
| Display | 28px (1.75rem) | 600 (semibold) | 1.2 | Hero card title, page heading "Blog" |

**2 weights** declared: regular (400), semibold (600).

Prose content headings (h1-h6) within MDX-rendered blog posts use `@tailwindcss/typography` plugin defaults, not these tokens.

## Color

| Role | Token | Usage |
|------|-------|-------|
| Dominant (60%) | `--background` | Page background, main surfaces |
| Secondary (30%) | `--card` | Blog cards, sidebar TOC, series banner background |
| Accent (10%) | `--accent` (cyan oklch) | Active tag pill, breadcrumb link hover, share icon hover, active TOC heading indicator, links, focus rings |
| Accent-secondary | `--accent-secondary` (violet oklch) | Gradient fallback secondary endpoint (combined with --accent for cover image fallbacks) |
| Muted foreground | `--muted-foreground` | Secondary text (dates, reading time, breadcrumb separators, inactive TOC items) |
| Border | `--border` | Card borders, separators |
| Foreground | `--foreground` | Primary text |

**Accent reserved for:** tag pill active variant, breadcrumb link hover, share button icon hover and focus, active TOC heading (left border + text color), post title link hover on cards, focus ring on interactive elements, gradient cover fallback endpoints.

## Component Inventory

### 1. BlogHeroCard — `src/components/blog/blog-hero-card.tsx`

Full-width hero card for the latest post.

- **Cover image:** Fills card background area using `object-fit: cover` with a dark overlay for text readability
- **Fallback:** `--gradient-hero` (cyan to violet linear gradient 135deg), same treatment as hero section
- **Content:** Title (Display token), excerpt (Body token, 3-line clamp), date + reading time (Small token), tags (Badge pills)
- **Max-height:** 480px on desktop, auto on mobile
- **Interaction:** Entire card is clickable -> links to `/blog/[slug]`
- **Hover:** `transform: scale(1.01)`, shadow `shadow-md` -> `shadow-lg`, border accent highlight
- **Focus:** Outline ring `ring-2 ring-accent`
- **Active:** `transform: scale(0.99)`, brief

### 2. BlogCard — `src/components/blog/blog-card.tsx`

Grid card for non-hero posts. Reuses existing `Card` compound component from shadcn/ui.

- **Cover thumbnail:** 16:9 aspect ratio container with `object-fit: cover`, gradient fallback
- **Content:** Title (Heading token), excerpt (Body token, 2-line clamp via `line-clamp-2`), date + tags (Small token)
- **Interaction:** Entire card is clickable -> links to `/blog/[slug]`
- **Hover:** Border highlight (--accent), slight lift (translateY(-2px)), shadow increase
- **Focus:** Outline ring `ring-2 ring-accent`
- **Active:** `transform: scale(0.99)`

### 3. BlogList — `src/components/blog/blog-list.tsx`

Composition orchestrator for the blog list page.

- **Hero slot:** Renders `BlogHeroCard` with `getAllPosts()[0]`
- **Grid slot:** Renders remaining posts in responsive grid using `BlogCard`
- **Responsive grid:** 2 columns at >=768px, 1 column below
- **Tag filter row:** `TagFilter` component above the grid
- **Filter behavior:** Reads `?tag=` from URL search params, filters posts client-side
- **Transition on filter:** Content fades (opacity 0->1, 300ms, ease-out) wrapped in a suspense-like keyed wrapper
- **Empty filter state:** Shows empty state from copywriting contract

### 4. TagFilter — `src/components/blog/tag-filter.tsx`

Tag pill filtering row.

- **Source:** Tags from `getAllTags()`
- "All" pill as first option (clears filter, shows all posts)
- Uses existing `Badge` component:
  - Inactive: `variant="outline"` 
  - Active: `variant="default"` (accent background), or `variant="secondary"`
  - Active tag should have a subtle accent-colored left border or bg to distinguish
- **Interaction:** Clicking a tag sets `?tag=` URL param via `router.push` (client-side navigation)
- **Scroll behavior:** Clicking a tag scrolls to the post grid section (not page top)
- **Transition:** Tag filter change uses 300ms ease-out opacity + translateY fade
- **Responsive:** Horizontal scroll on mobile if many tags, with fade overflow indicator

### 5. Breadcrumb — `src/components/blog/breadcrumb.tsx`

- **Trail format:** `Blog > [tag or "Posts"] > [title]`
- **Logic:**
  - First segment: "Blog" links to `/blog`
  - Middle segment: first tag from `post.tags` if exists, else "Posts" — if it's a tag, links to `/blog/tag/[tag]`
  - Last segment: post title, plain text (not linked), truncated at ~40 chars
- **Separator:** "/" character in `text-muted-foreground`, with `px-2` spacing
- **Typography:** Small token, muted-foreground throughout, last item in foreground
- **Hover on links:** Color -> accent

### 6. TableOfContents — `src/components/blog/toc.tsx`

Adaptive TOC: sticky sidebar on desktop, collapsible toggle on mobile/tablet.

**Desktop (>=1024px):**
- Sticky sidebar on right side of article
- `position: sticky; top: 96px` (below fixed header)
- Width: ~220px
- Title: "On this page" in Small token, semibold, muted foreground
- TOC items: h2 and h3 headings
  - h2: flush left
  - h3: indented 12px left, slightly smaller
  - Each item: anchor link to heading `id`, Small token, 400 weight
- Active item: accent color + 2px left border in accent color
- Inactive items: muted foreground
- Hover on items: color transitions to accent

**Mobile/tablet (<1024px):**
- Collapsible section at top of article
- Toggle button: "On this page" with chevron icon (lucide `ChevronDown`)
- Collapsed by default
- Expand animation: max-height + opacity, 300ms ease-out
- Same heading hierarchy as desktop version

**Technical behavior:**
- Auto-generated from `headings: { id, text, level }[]` array (pre-computed server-side in `getPostBySlug` and passed as prop)
- Active heading tracking via IntersectionObserver with `rootMargin: "-20% 0px -60% 0px"` and thresholds `[0, 0.25, 0.5]` -- matching existing `useActiveSection` hook pattern
- Clicking a TOC item smoothly scrolls to the heading: target element needs `scroll-margin-top: 96px`

### 7. SeriesNav — `src/components/blog/series-nav.tsx`

Series navigation banner, rendered at both top and bottom of article.

- **Condition:** Only rendered when `post.series` is defined
- **Content:** "Part N of M -- Series Title"
  - N = position of current post within series (1-indexed, posts sorted by date)
  - M = total posts in series
  - Series title from frontmatter
- **Visual:** 
  - Border-left: 3px solid `--accent`
  - Background: `--muted` / `--card` (subtle)
  - Rounded corners: `rounded-md`
  - Typography: Small token, muted-foreground text, series title in foreground
  - Padding: `p-3` (12px)
- **Navigation:** Left arrow (older in series) + Right arrow (newer in series) using lucide `ChevronLeft` / `ChevronRight`
- **Arrows:** Only rendered if prev/next part exists in series

### 8. ShareButtons — `src/components/blog/share-buttons.tsx`

Row of share action buttons at the bottom of article content, between series footer nav and prev/next nav.

- **Platforms:** X/Twitter, LinkedIn, Copy link
- **X:** `https://twitter.com/intent/tweet?text={encoded title}&url={encoded url}`
- **LinkedIn:** `https://linkedin.com/sharing/share-offsite/?url={encoded url}`
- **Copy link:** `navigator.clipboard.writeText(url)` -> shows "Copied!" tooltip for 2 seconds
- **Visual:**
  - Icon-only with accessible `sr-only` label text
  - Icons: use lucide icons (`ExternalLink` for share destinations, `Link` or `Copy` for copy link) -- or lucide's brand icons if available
  - Muted foreground, `hover:text-accent` with `hover:scale-110`
  - Focus: outline ring
  - Active: scale 0.95
- **Layout:** Centered row with `gap-4` between icons

### 9. PrevNextNav — `src/components/blog/prev-next-nav.tsx`

Full-width adjacent post navigation at the bottom of the post page.

- **Layout:** Two columns -- "Previous" (older post) on left, "Next" (newer post) on right
- **Content:** Directional label ("Previous" / "Next") in Small token muted-foreground + post title in Body token foreground
- **Arrows:** Lucide `ChevronLeft` on left side, `ChevronRight` on right side
- **Series-bound:** If `post.series` is defined, only show posts from the same series (sorted by date)
- **Fallback:** Chronological adjacency via `getAllPosts()` when no series
- **Empty side:** If no neighbor exists on one side, that column is empty (or rendered as a spacer)
- **Hover:** Title shifts toward arrow (left arrow moves title right, right arrow moves title left), accent color
- **Focus:** Outline ring
