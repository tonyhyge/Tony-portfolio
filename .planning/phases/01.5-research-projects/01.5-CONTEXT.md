# Phase 1.5: Research Projects — Context

**Gathered:** 2026-06-21
**Status:** Ready for planning

<domain>
## Phase Boundary

Deliver a research projects showcase section featuring Minh's academic/ML research. Grid of project cards with status badges (in-progress/submitted/published/accepted), abstract/summary, tech tags, paper links (arXiv/DOI/PDF), and status-based filter. Designed to reflect the academic (40%) side of the portfolio's Modern + Academic identity.

**In scope:** Card grid, status badges, filter tabs, paper links, data model for research entries, empty state.
**Not in scope:** Blog integration, publication citation manager, lab profile page (Phase 3.x), interactive demos (Phase 3.3).
</domain>

<decisions>
## Implementation Decisions

### Layout Grid
- **D-01:** Bento grid — 2 cột trên desktop, 1 cột mobile
- **D-02:** Featured project chiếm 2 cột (dạng hero card)
- **D-03:** Featured chọn bằng manual flag trong data model (`featured: true`)
- **D-04:** Trên mobile featured card vẫn nổi bật hơn (visual hierarchy, không giãn kích thước)

### Status Badges
- **D-05:** Custom colors per status (mở rộng Badge component):
  - `in-progress` — amber/warm tone
  - `submitted` — blue
  - `published` — green
  - `accepted` — purple
- **D-06:** Badge dùng variant mới, không phải `Badge` variant có sẵn (custom variant set)

### Paper Links
- **D-07:** Hiển thị dạng icon chips (icon + label) — gọn, dễ thấy
- **D-08:** Các link hỗ trợ: arXiv, DOI, PDF (optional)

### Filter by Status
- **D-09:** Tabs/chips filter sử dụng URL search param (`?status=`) — shareable
- **D-10:** Auto-detect tabs — chỉ hiển thị status có dự án (tính từ data)
- **D-11:** Crossfade animation khi chuyển filter

### Data Model
- **D-12:** TypeScript data file (`src/data/research.ts`) — không dùng MDX
- **D-13:** Dùng Zod schema để validate + auto-generate type
- **D-14:** Fields:
  - `title` — string
  - `status` — enum: `in-progress | submitted | published | accepted`
  - `tags` — string[]
  - `abstract` — string
  - `links` — object with optional: `arxiv`, `doi`, `pdf`
  - `startDate` — string (ISO date)
  - `endDate` — string (ISO date, optional for in-progress)
  - `venue` — string (conference/journal name, optional)
  - `collaborators` — string[] (co-authors, optional)
  - `featured` — boolean (manual flag, default false)
  - `thumbnail` — string (path to figure/diagram image, optional)

### Visual / Card Content
- **D-15:** Mỗi card có diagram/figure từ paper (thumbnail optional)
- **D-16:** Abstract truncated 3-4 dòng + expand/collapse

### Empty State
- **D-17:** Message + suggestion: "No {status} projects yet" + button "Show all"

### Claude's Discretion
- CSS animation timing và easing cho crossfade filter transition
- Responsive breakpoint cụ thể cho grid collapse
- Icon library cho paper link chips (lucide-react có sẵn)

</decisions>
