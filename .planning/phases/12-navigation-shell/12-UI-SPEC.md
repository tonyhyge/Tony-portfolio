## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, dot indicator offset |
| sm | 8px | Nav link internal padding, icon-to-label gaps |
| md | 16px | Header inner padding, footer padding |
| lg | 24px | Header content gap (logo \| nav \| toggle \| hamburger) |
| xl | 32px | Footer content spacing |
| 2xl | 48px | Not used in this phase |
| 3xl | 64px | Not used in this phase |

**Header-specific spacing:**
- Header height: `h-14` (56px) — touch-target compliant
- Header inner padding: `px-4 sm:px-6 lg:px-8` — matches container pattern
- Nav link hit area: `h-9` (36px) minimum — from existing `nav-links.tsx`

**Exceptions:** Touch targets on mobile Sheet nav links use `min-h-[44px]` (WCAG 2.5.5 touch target size, exceeds standard 8-point scale).

**Source:** `11-PATTERNS.md` (container spacing, nav link dimensions), existing `nav-links.tsx` (`h-9`), NFR3 (touch-friendly)

**Notes:**

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `--background` (`oklch(1 0 0)` light, `oklch(0.145 0 0)` dark) | Header transparent at top, background surfaces |
| Secondary (30%) | `--background/80` with `backdrop-blur-md` | Header glassmorphism state after scroll |
| Accent (10%) | `--accent` (`oklch(0.65 0.18 210)`) | Active section dot indicator, mobile hamburger icon in open state |
| Destructive | `--destructive` (`oklch(0.577 0.245 27.325)` light, `oklch(0.704 0.191 22.216)` dark) | Not used in this phase — reserved for future destructive actions |

**Accent reserved for:**
- Active section dot indicator (the single accent-colored element in the header)
- Mobile menu Sheet trigger icon (only when Sheet is open — `text-accent` state)

**Header backdrop state:**
- At scroll position 0: `bg-transparent` (fully transparent, no backdrop)
- At scroll > 50px: `bg-background/80 backdrop-blur-md border-border/50` (glassmorphism)
- Transition: `transition-all duration-200` (smooth, not instant)

**Header hide/show state:**
- Scroll down (hidden): `transform -translate-y-full`
- Scroll up (visible): `transform translate-y-0`
- Transition: `transition-transform duration-200 ease-in-out`

**Source:** `globals.css` (--accent, --destructive, --background tokens), `12-CONTEXT.md` D-02 (backdrop fade at ~50px), `sticky-header.tsx` (existing glassmorphism classes), `11-CONTEXT.md` (accent palette)

## Interaction States

### Header States

| State | Condition | Visual |
|-------|-----------|--------|
| Default (top) | `scrollY === 0` | Transparent bg, visible content, no border |
| Scrolled (up) | `scrollY > 50` AND direction === up | `bg-background/80`, `backdrop-blur-md`, `border-b border-border/50`, visible |
| Scrolled (down) | `scrollY > 50` AND direction === down | Same as scrolled up, but `transform -translate-y-full` (hidden) |
| Scrolled (top, after hidden) | User scrolls back up past 50px threshold | `transform translate-y-0` (reappears), glassmorphism active |

### Mobile Menu States

| State | Condition | Visual |
|-------|-----------|--------|
| Closed | Sheet not open | Hamburger icon visible in header |
| Open | Sheet triggered | Sheet slides in from right, overlay at `bg-black/10` with `backdrop-blur-xs`, close button top-right |

### Nav Link States

| State | Condition | Visual |
|-------|-----------|--------|
| Default | Not active | `text-muted-foreground`, no dot |
| Hover/Focus | Pointer hover or keyboard focus | `bg-muted`, `focus-visible:ring-3 focus-visible:ring-ring/50` |
| Active | Section in view (IntersectionObserver) | `text-foreground` + accent dot below or beside label |

## Accessibility Contract

| Requirement | Implementation |
|-------------|----------------|
| Keyboard navigation | Tab order: logo → nav links (left to right) → theme toggle → hamburger (mobile only). All nav links reachable via Tab. |
| Focus visible | `focus-visible:ring-3 focus-visible:ring-ring/50` on all nav links (from existing `nav-links.tsx`). |
| ARIA labels | `aria-label="Main navigation"` on nav element. `aria-label="Toggle mobile menu"` on hamburger trigger. `aria-expanded` on hamburger (tracks Sheet open state). |
| Screen reader | Header uses `<header>` semantic element. Footer uses `<footer>`. Nav uses `<nav>`. Active section dot is `aria-hidden="true"` (visual-only indicator, active state conveyed via text change). |
| Reduced motion | Lenis respects `prefers-reduced-motion` — disables smooth scroll animation. Header hide/show animation disabled (header stays visible). CSS `motion-reduce:animate-none` on all animated elements. |
| Color contrast | All text meets WCAG AA (4.5:1 minimum). Accent dot is supplementary — information not conveyed by color alone (active link also changes text weight/color). |
| Touch targets | Mobile nav links: `min-h-[44px]`. Hamburger icon button: `size-10` (40px) minimum. |
