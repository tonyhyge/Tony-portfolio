# Phase 01.6: Skills & Contact — Context

**Gathered:** 2026-06-21
**Status:** Ready for planning

<domain>
## Phase Boundary

Skills section (categorized tech tags, accordion) + Contact section (Formspree form, social links) on the single-page portfolio. Includes: accordion-style skill categories with per-category color differentiation, skill → project cross-linking via custom popover, contact form (Name/Email/Message) with Formspree backend and inline confirmation, social links row (GitHub, LinkedIn, Google Scholar, Email).

**In scope:** Accordion categories, skill tags with per-category colors, custom popover for skill→project linking, Formspree contact form with validation, inline confirmation after submit, social icon row, visual differentiation for section headings.
**Not in scope:** Blog integration (M2), publication citation manager (future), interactive skill demonstrations (Phase 3.x).
</domain>

<decisions>
## Implementation Decisions

### Skill Level Display
- **D-01:** Tags thuần (string[]), không visual indicator cho proficiency level
- **D-02:** Proficiency được chứng minh qua các section khác (Research papers, Experience roles, About stats) — không self-assessed progress bars

### Skills Section Layout
- **D-03:** Accordion categories — single open (mở 1 category, auto-close cái khác)
- **D-04:** Expand/collapse dùng JS height calculation (useRef + useEffect, scrollHeight + max-height transition)

### Skill Tags Visual Treatment
- **D-05:** Per-category color differentiation — mỗi category dùng variant badge riêng:
  - Machine Learning & Deep Learning — cyan
  - Programming Languages — violet
  - Tools & Frameworks — amber
  - (Research Methods — đã bỏ, xem D-08)

### Skill Data Model — Cross-Category
- **D-06:** Mỗi skill có thể appear ở nhiều categories (cross-category tags)
- **D-07:** Data model mở rộng:
  ```typescript
  interface Skill {
    name: string
    categories: string[]  // belongs to multiple categories
    relatedProjects?: { slug: string; title: string }[]
  }
  ```

### Research Methods Category
- **D-08:** Bỏ category "Research Methods" (Experimental Design, Statistical Analysis, Paper Writing, Literature Review, Benchmarking) — skills đã được thể hiện qua Experience (Research Assistant role) và Research papers

### Skill → Project Linking
- **D-09:** Click skill tag → custom popover (gần tag, không phải shadcn Sheet) hiển thị danh sách project titles
- **D-10:** Click project title → smooth scroll tới Research section
- **D-11:** Data: `relatedProjects: { slug: string; title: string }[]` — lưu slug + title để hiển thị nhanh, không cần query research data

### Form Backend
- **D-12:** Formspree (free plan) — thêm action URL vào form HTML
- **D-13:** 3 fields: Name + Email + Message
- **D-14:** Client + Server validation — không cần format quốc tế đặc biệt
- **D-15:** Spam protection: honeypot field (Formspree built-in), không reCAPTCHA

### Form UX
- **D-16:** Inline confirmation — form biến mất, thay bằng confirmation message: "Thanks, Minh will get back to you soon"
- **D-17:** Error validation inline per-field — highlight field + message dưới field lỗi

### Social Links
- **D-18:** Core 4 links — GitHub, LinkedIn, Google Scholar, Email
- **D-19:** Icon row, không label. Email icon → `mailto:` link
- **D-20:** Không location text hay availability status

### Contact Section Layout
- **D-21:** Split layout — form (left) + social links (right) trên desktop. Stack trên mobile: form trước, social links dưới.

### Section Headings
- **D-22:** Visual differentiation — Skills heading dùng gradient text effect (`bg-gradient-to-r`), Contact heading có icon accent. Không dùng Section component generic style.

### Animations
- **D-23:** Skills accordion: expand/collapse với height transition (JS scrollHeight)
- **D-24:** Contact section: staggered fade-in cho form fields + social icons khi scroll vào viewport

### Claude's Discretion
- Accordion chevron animation direction và rotation
- Popover exact styling (kích thước, position relative tới tag, close mechanism)
- Stagger delay timing cho fade-in animation (default 50-100ms stagger)
- Form input styling (border, focus ring, label placement) — consistent với design system
- Social icon hover effects (scale + color shift)
- Gradient text implementation cho Skills heading (Tailwind utilities vs inline style)

</decisions>

<specifics>
## Specific Ideas

- **Accordion:** Single-open pattern. JS scrollHeight calculation for smooth expand/collapse. Chevron icon rotates on open. Category title có accent color dot matching tag color.
- **Skill tags as clickable chips:** Hover → tooltip-style indicator "View related projects". Click → popover near tag.
- **Popover:** Nhỏ gọn — danh sách project titles dạng links. Click → smooth scroll tới Research section. Close khi click outside hoặc Escape.
- **Formspree:** `action="https://formspree.io/f/{id}"` + `method="POST"`. Hidden `_gotcha` field for honeypot. Server validates + forwards to email.
- **Contact split layout:** CSS grid `grid-cols-2` desktop → `flex-col` mobile. Form padding-right, social icons centered right column.
- **Section visual differentiation:** Skills heading dùng gradient text (cyan→violet giống hero name). Contact heading có Mail icon accent beside title.
