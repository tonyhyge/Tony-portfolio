    Create `src/hooks/useClickOutside.ts` — a custom hook that:
    - Takes a callback function `handler: () => void`
    - Returns a `ref` of type `RefObject<HTMLElement | null>`
    - Uses `useRef` for the element reference
    - In `useEffect`, attaches `mousedown` and `touchstart` listeners to `document`

    - The `handler` dependency is stable via useCallback on the consumer side (document in action, not in hook)
    - Export the generic function `useClickOutside`
