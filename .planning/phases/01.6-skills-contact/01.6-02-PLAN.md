  <action>
    Create `src/components/skills/SkillPopover.tsx` as a `"use client"` component.

    **Props:**
    - `projects: { slug: string; title: string }[]` — the related projects to show
    - `onClose: () => void` — close callback (for click-outside and Escape)
    - `triggerRef: React.RefObject<HTMLElement | null>` — reference to the tag that was clicked, for positioning
    - `isOpen: boolean` — visibility control

    **Positioning:**
    - Render near the trigger element. Use a fixed-position approach:
      - On mount (or when `isOpen` becomes true), read `triggerRef.current.getBoundingClientRect()`
      - Position the popover below and left-aligned with the trigger (or right-aligned if near screen edge)
      - Use inline `style={{ position: 'fixed', top: ..., left: ... }}` so it's not constrained by parent overflow
    - Set `zIndex: 50` to appear above other content
    - Minimum width: ~200px, maximum: trigger width or ~280px

    **Content:**
    - A small container with `bg-popover` background, `border border-border`, rounded corners, shadow
    - Header: small label "Related projects" in muted text
    - List: the project titles as clickable items
    - Each item is a `<button>` (not a link — we navigate programmatically per D-10)
    - `onClick` → `onClose()`, then `document.getElementById('research')?.scrollIntoView({ behavior: 'smooth' })`
    - If no projects in the array, show a fallback: "Learn more through my projects" in muted text

    **Dismissal:**
    - Use `useClickOutside(onClose)` from the shared hook — pass the popover container ref from the hook's returned ref
    - Use `useEffect` with `keydown` listener for Escape key → `onClose()`
    - Cleanup both listeners on unmount

    **Accessibility:**
    - `role="dialog"` on the popover container
    - `aria-label="Related projects"`
    - Focus management: auto-focus a close or first-item button when opened

    **Styling:**
    - Padding: `p-3`
    - List items: `py-1.5 px-2 rounded-md hover:bg-muted cursor-pointer text-sm`
    - Header: `text-xs text-muted-foreground mb-1.5`
    - Add a subtle entrance animation (optional, Claude's discretion): a quick fade-in with `opacity-0 → opacity-100 transition-opacity duration-150`
