## Spacing Scale

Declared values (Tailwind 8-point scale, multiples of 4):

| Token | px | rem | Usage |
|-------|----|------|-------|
| xs | 4px | 0.25rem | Inline padding, icon gaps |
| sm | 8px | 0.5rem | Compact element spacing, button padding |
| md | 16px | 1rem | Default element spacing, horizontal padding |
| lg | 24px | 1.5rem | Section padding, card inner padding |
| xl | 32px | 2rem | Layout gaps between cards, section title to content |
| 2xl | 48px | 3rem | Major section breaks (`py-12`) |
| 3xl | 64px | 4rem | Page-level whitespace, section-to-section gaps (`py-16` up to `py-24`) |

Exceptions: None for MVP.

## Typography

| Role | Tailwind Token | px | rem | Weight | Line Height |
|------|---------------|-----|------|--------|-------------|
| Label | `text-sm` | 14px | 0.875rem | 400 (regular) | 1.5 |
| Body | `text-base` | 16px | 1rem | 400 (regular) | 1.5 |
| Section heading | `text-3xl` (mobile), `text-4xl` (md+) | 30px / 36px | 1.875rem / 2.25rem | 600 (semibold) | 1.2 |
| Hero display | `text-5xl` | 48px | 3rem | 600 (semibold) | 1.2 |

Notes:
- **Headings:** Inter at weight 600. Heading line-height 1.2.
- **Body:** Inter at weight 400. Body line-height 1.5.
- **Code/mono:** JetBrains Mono at weight 400. Use for inline `<code>` and code blocks.
- **Semibold body emphasis:** Weight 500 (Inter). Undefined weight: weight 700 reserved for hero name emphasis.
- Responsive heading sizes: section headings bump from `text-3xl` (30px) to `text-4xl` (36px) at `md:` breakpoint.

## Color

Existing shadcn default palette already defined in `globals.css` (`:root` and `.dark`). Phase 1.1 overrides `--accent` / `--accent-foreground` and adds new tokens for the portfolio's cyan-teal/violet identity.

| Role | Value (light mode) | Value (dark mode) | Usage |
|------|--------------------|--------------------|-------|
| Dominant (60%) | `--background: oklch(1 0 0)` | `--background: oklch(0.145 0 0)` | Page background, primary surface |
| Secondary (30%) | `--card: oklch(1 0 0)`, `--secondary: oklch(0.97 0 0)` | `--card: oklch(0.205 0 0)`, `--secondary: oklch(0.269 0 0)` | Cards, nav, supporting surfaces |
| Accent (10%) | `--accent: oklch(0.65 0.18 210)` (cyan-teal) | `--accent: oklch(0.65 0.18 210)` (unchanged) | CTAs, active nav links, inline links, focus rings, hero gradient |
| Accent secondary | `--accent-secondary: oklch(0.55 0.15 280)` (violet) | `--accent-secondary: oklch(0.6 0.18 280)` (brighter) | Hero gradient complementary, decorative accents |
| Destructive | `--destructive: oklch(0.577 0.245 27.325)` | `--destructive: oklch(0.704 0.191 22.216)` | Not used in Phase 1.1 (defined for future phases) |

### New CSS Custom Properties to Add in `globals.css`

```css
:root {
  --accent: oklch(0.65 0.18 210);
  --accent-foreground: oklch(0.98 0 0);
  --accent-secondary: oklch(0.55 0.15 280);
  --gradient-hero: linear-gradient(135deg, oklch(0.65 0.18 210), oklch(0.55 0.15 280));
}

.dark {
  --accent: oklch(0.65 0.18 210);
  --accent-foreground: oklch(0.98 0 0);
  --accent-secondary: oklch(0.6 0.18 280);
}
```

### `@theme inline` Block Updates

Add to the existing block in `globals.css`:

```css
@theme inline {
  --color-accent: var(--accent);
  --color-accent-foreground: var(--accent-foreground);
  --color-accent-secondary: var(--accent-secondary);
  --font-heading: var(--font-inter);
}
```

Accent reserved for: CTA buttons ("Download CV"), active/current navigation link, inline links, focus ring outlines, hero gradient primary color, decorative accent borders on emphasis elements, icon accent fills.

## Interaction States

| Element | States | Behavior |
|---------|--------|----------|
| Theme toggle button | idle, hover, focus-visible, active | Renders `<div className="size-8" />` placeholder until mounted to prevent hydration mismatch. On click: toggles `.dark` on `<html>`, writes to `localStorage("theme")`. |
| Nav link | default, hover, active (current section), focus-visible | Smooth-scroll to section anchor on click. Active state tracked later via IntersectionObserver (deferred to Phase 1.2). |
| CTA button (Download CV) | default, hover, focus-visible, active | shadcn Button default variant. Opens PDF or triggers download. |
| Hero gradient | idle (animated) | CSS `@keyframes gradient-shift` animates `background-position`. `@media (prefers-reduced-motion: reduce)` pauses animation. |
| Sticky header | default, scrolled | On scroll: add `shadow-sm` and `backdrop-blur` background. Use `position: sticky; top: 0`. |
